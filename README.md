# sticker-tracker
Track sticker usage in a Discord server.

## Setting up

### Dependencies

* discord.js
* quick.db
* better-sqlite3

### Running

Clone the repository:
```
git clone https://github.com/okadi/sticker-tracker.git
```

Enter directory:
```
cd sticker-tracker
```

Install dependencies:
```
npm install
```

Start the bot:
```
npm start
```
